## Read-only website

GitHub Pages: https://doview.github.io/obsidian-pickle-tracking-list/

### Local dev

```bash
cd site
npm install
npm run dev
```

### How it works

The site is a static SvelteKit app that parses `Pickles Tracking List.md` at build time and renders a searchable/filterable catalog.
